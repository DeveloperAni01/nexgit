## рҹҳұ The Problem

Every fresher joining Wipro, TCS, Infosys, or any bootcamp struggles with Git daily:

- вқҢ Don't know the difference between local Git and GitHub
- вқҢ Accidentally push `node_modules`, `.env`, `bin`, `obj` files
- вқҢ Merge wrong branches вҖ” everything breaks
- вқҢ Git errors look like alien language
- вқҢ No tool explains what went wrong in plain English

**NexGit fixes ALL of that.** One tool. Zero confusion.

---

## вңЁ Features

- рҹӨ– **AI Powered** вҖ” Explains errors, generates commit messages, writes README files
- рҹҡҖ **One Command Setup** вҖ” `nexgit init` creates local repo + GitHub repo + .gitignore + README automatically
- рҹӣЎпёҸ **Safety First** вҖ” Warns before every dangerous action
- рҹ“қ **Smart Commits** вҖ” AI suggests perfect conventional commit messages
- рҹҢҚ **Multi Language** вҖ” English, Hinglish, Hindi support
- рҹҺҜ **Beginner Friendly** вҖ” No Git knowledge required
- вҡЎ **Works Everywhere** вҖ” Windows, Mac, Linux


## рҹҡҖ Quick Start

```bash
# 1. One time setup
nexgit setup

# 2. Create a new project with GitHub repo
nexgit init

# 3. Check what changed
nexgit status

# 4. Commit with AI generated message
nexgit commit

# 5. Push to GitHub
nexgit push
```

## рҹ“– All Commands

### рҹ”§ Setup & Init

| Command | Description |
|---------|-------------|
| `nexgit setup` | One time setup вҖ” language + GitHub token |
| `nexgit init` | Create local repo + GitHub repo + .gitignore + AI README |

### рҹ“Ҡ Daily Workflow

| Command | Description |
|---------|-------------|
| `nexgit status` | See git state in plain English |
| `nexgit commit` | Smart commit with AI message suggestion |
| `nexgit commit "message"` | Commit with your own message |
| `nexgit push` | Safely push to GitHub with warnings |
| `nexgit pull` | Safely pull latest changes |

### рҹҢҝ Branch Management

| Command | Description |
|---------|-------------|
| `nexgit branch` | List all branches |
| `nexgit branch "name"` | Create a new branch |
| `nexgit switch "branch"` | Switch to another branch |
| `nexgit merge "branch"` | Safely merge with warnings |

### рҹӣ пёҸ Utilities

| Command | Description |
|---------|-------------|
| `nexgit diff` | See what changed in your files |
| `nexgit history` | See commit history in plain English |
| `nexgit ignore` | Auto fix your .gitignore file |
| `nexgit undo` | Safely undo git actions |
| `nexgit explain "error"` | AI explains any git error |
| `nexgit lang` | Change language (English/Hinglish/Hindi) |

## рҹӣЎпёҸ Safety Features

NexGit warns you before EVERY dangerous action:

- рҹҡЁ Pushing `.env`, `node_modules`, `bin`, `obj` files
- рҹҡЁ Merging directly into `main` or `master`
- рҹҡЁ Force pushing or deleting branches
- рҹҡЁ Discarding uncommitted changes
